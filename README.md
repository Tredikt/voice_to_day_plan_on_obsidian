# 🎙️ Voice to Day Plan Telegram Bot

Этот Telegram-бот принимает голосовые сообщения, преобразует их в текст, структурирует список ваших задач на день с помощью ИИ **DeepSeek** (или **OpenAI**) и автоматически синхронизирует его с вашим личным хранилищем **Obsidian**, используя **Yandex Disk (WebDAV)** или **Google Drive**.

> 📅 **Важно:** Интеграция с Google Календарем является **полностью опциональной**. Бот может работать в классическом режиме: просто распознавать голос и сохранять планы в ваш Obsidian без подключения какого-либо календаря.

---

## 📖 Полное руководство по настройке с нуля

Ниже приведено пошаговое руководство, которое поможет вам настроить и запустить этот проект на любом компьютере или сервере с нуля.

---

### Шаг 1. Создание Telegram-бота

1. Откройте Telegram и найдите официального бота **[@BotFather](https://t.me/BotFather)**.
2. Отправьте команду `/newbot` для создания нового бота.
3. Введите отображаемое имя бота (например, `Мой планировщик Obsidian`).
4. Введите уникальный юзернейм бота, оканчивающийся на `_bot` (например, `my_obsidian_planner_bot`).
5. **Скопируйте полученный API токен** (он выглядит как `8603827673:AAE4-0ST...`). Этот токен будет использоваться в переменной `TELEGRAM_BOT_TOKEN`.

---

### Шаг 2. Настройка распознавания речи (Speech-to-Text)

Вы можете выбрать один из двух вариантов распознавания голоса:

*   **Вариант А: Полностью бесплатное локальное распознавание (Рекомендуется)**
    *   **Не требует ключей и оплаты.** Бот автоматически использует встроенные библиотеки Python (`SpeechRecognition` и Google Web Speech API) и утилиту `FFmpeg` внутри Docker-контейнера для бесплатной конвертации и распознавания.
    *   *Настройка:* Оставьте переменную `OPENAI_API_KEY` пустой.
*   **Вариант Б: Облачный OpenAI Whisper API**
    *   *Требует:* Аккаунт OpenAI с положительным балансом и сгенерированный API-ключ.
    *   *Настройка:* Укажите ваш API-ключ в переменной `OPENAI_API_KEY`.

---

### Шаг 3. Настройка ИИ-генератора задач (DeepSeek или OpenAI)

Для анализа текста и составления списка To-Do задач на день бот использует современные текстовые нейросети:

1. **Режим по умолчанию — DeepSeek (Рекомендуется)**:
   * Самая качественная и невероятно дешевая генерация на русском языке.
   * Зарегистрируйтесь на [платформе DeepSeek API](https://platform.deepseek.com/).
   * Создайте API-ключ и вставьте его в переменную `DEEPSEEK_API_KEY`.
2. **Альтернативный режим — OpenAI**:
   * Если у вас нет ключа DeepSeek, но указан `OPENAI_API_KEY`, бот автоматически переключится на высокоэффективную модель `gpt-4o-mini` от OpenAI.

---

### Шаг 4. Настройка облачного хранилища Obsidian

Выберите, куда именно вы хотите сохранять файлы заметок Obsidian:

#### Вариант А: Яндекс Диск (Рекомендуется)
1. Зайдите в ваш аккаунт Яндекс Диска.
2. Создайте в корне Диска папку, где лежит ваше хранилище Obsidian (например, `/Obsidian/Day plans`).
3. Зайдите на страницу безопасности **[Яндекс ID](https://id.yandex.ru/)**.
4. Перейдите в раздел **«Безопасность»** -> **«Пароли приложений»**.
5. Нажмите **«Создать новый пароль»**, выберите тип **WebDAV** и дайте ему любое имя (например, `Obsidian Bot`).
6. Скопируйте сгенерированный пароль.
7. Заполните переменные:
   * `STORAGE_PROVIDER=yandex`
   * `YANDEX_USER=ваш_email@yandex.ru`
   * `YANDEX_PASSWORD=сгенерированный_пароль_приложения`
   * `YANDEX_OBSIDIAN_DIR=/Obsidian/Day plans` (папка, где бот будет создавать ежедневные заметки формата `YYYY-MM-DD.md`).

#### Вариант Б: Google Drive
1. Создайте проект в **[Google Cloud Console](https://console.cloud.google.com/)**.
2. Включите **Google Drive API** для вашего проекта.
3. Перейдите в раздел **Credentials** (Учетные данные) и создайте **Service Account** (Сервисный аккаунт).
4. Нажмите на созданный сервисный аккаунт, перейдите во вкладку **Keys** (Ключи) -> **Add Key** -> **Create New Key** (формат **JSON**). Скачанный JSON-файл переименуйте в `google-credentials.json` и положите его в корневую папку бота.
5. Скопируйте email-адрес вашего Сервисного аккаунта (он выглядит как `my-service-account@project.iam.gserviceaccount.com`).
6. Перейдите в ваш Google Диск, найдите или создайте папку для ежедневных планов Obsidian, нажмите на нее правой кнопкой мыши -> **Поделиться** -> и предоставьте права **«Редактор»** для email-адреса сервисного аккаунта.
7. Заполните переменные:
   * `STORAGE_PROVIDER=google`
   * `GOOGLE_CREDENTIALS_JSON=google-credentials.json`
   * `GOOGLE_OBSIDIAN_DIR_ID=уникальный_id_вашей_папки_из_адресной_строки_браузера`

---

### Шаг 5. Настройка Google Календаря (Полностью опционально)

Если вы хотите, чтобы задачи с конкретным указанием времени (например: *"Потренироваться сегодня в 18:00"*) автоматически добавлялись в ваш календарь:

1. В том же проекте **Google Cloud Console** найдите и включите **Google Calendar API**.
2. Откройте ваш личный **[Google Календарь](https://calendar.google.com/)** в браузере.
3. Наведите мышку на нужный календарь в левой панели, нажмите на три точки -> **«Настройки и общий доступ»**.
4. В разделе **«Доступ для отдельных пользователей»** нажмите **«Добавить»** и вставьте email вашего Сервисного аккаунта (из Шага 4). Назначьте ему права: **«Внесение изменений в мероприятия»**.
5. Прокрутите ниже до раздела **«Интеграция календаря»** и скопируйте **Идентификатор календаря** (обычно это ваш личный Gmail-адрес).
6. Заполните переменные:
   * `GOOGLE_CALENDAR_ID=ваш_календарь@gmail.com`
   * `TIMEZONE=Europe/Moscow` (укажите ваш часовой пояс)

> 💡 Если вы оставите переменную `GOOGLE_CALENDAR_ID` пустой, бот просто отключит этот функционал и будет работать только с Obsidian.

## 🚀 Быстрый запуск проекта через Docker Compose

### 1. Клонирование репозитория
Склонируйте репозиторий в любую удобную рабочую папку на вашем сервере или компьютере:
```bash
git clone https://github.com/Tredikt/voice_to_day_plan_on_obsidian.git
cd voice_to_day_plan_on_obsidian
```

### 2. Настройка конфигурационного файла `.env`
Создайте файл `.env` на основе примера:
```bash
cp .env.example .env
```
Откройте созданный `.env` файл в любом текстовом редакторе и заполните его:

```env
TELEGRAM_BOT_TOKEN=ваш_telegram_api_токен
TELEGRAM_PROXY=socks5://user:pass@host:port     # Опционально, если нужен прокси

OPENAI_API_KEY=                                # Опционально (оставьте пустым для бесплатного распознавания)
DEEPSEEK_API_KEY=ваш_api_ключ_deepseek         # Получите на platform.deepseek.com

# Укажите ваш ID в Telegram, чтобы посторонние люди не могли пользоваться вашим ботом.
# Ваш ID можно узнать в Telegram через бота @userinfobot. Несколько ID укажите через запятую.
ALLOWED_TELEGRAM_IDS=123456789

# Хранилище: yandex или google
STORAGE_PROVIDER=yandex

# Настройки для Яндекс Диска (WebDAV)
YANDEX_USER=ваш_логин@yandex.ru
YANDEX_PASSWORD=ваш_пароль_приложения_webdav
YANDEX_OBSIDIAN_DIR=/Obsidian/Day plans

# Настройки для Google Drive (если STORAGE_PROVIDER=google)
GOOGLE_CREDENTIALS_JSON=google-credentials.json
GOOGLE_OBSIDIAN_DIR_ID=

# Настройки для Google Календаря (Опционально)
GOOGLE_CALENDAR_ID=
TIMEZONE=Europe/Moscow
```

### 3. Сборка и запуск контейнера
Запустите бота в фоновом режиме одной простой командой:
```bash
docker-compose up -d --build
```

### 4. Проверка логов и статуса
Убедитесь, что бот успешно запустился и подключился к Telegram:
```bash
docker-compose ps
docker-compose logs -f
```

## 🎙️ Как пользоваться ботом

1. Зайдите в Telegram к вашему боту и нажмите кнопку **`Старт`** (или отправьте `/start`).
2. Запишите и отправьте голосовое сообщение с планами. Например:
   > *"Привет! Поставь мне задачу на сегодня: в 18:00 сходить на тренировку, а еще нужно обязательно позвонить маме и купить хлеб."*
3. **Что сделает бот:**
   * Скачает аудио, переведет его в текст (бесплатно локально или через Whisper).
   * ИИ проанализирует текст, составит красивый Obsidian Markdown To-Do список.
   * Сохранит или аккуратно допишет данные в файл **`YYYY-MM-DD.md`** в вашем облаке (Obsidian синхронизирует его на ваш ПК).
   * Если настроен Google Календарь, бот мгновенно создаст событие *"Сходить на тренировку"* на сегодня на 18:00.
   * Пришлет вам красивый структурированный текстовый отчет в ответ на голосовое сообщение!
